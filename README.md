# House Finder
You can easily find ideal house using this.

## Install

  ```
  $ git clone https://github.com/drumgiovanni/house_finder.git
  $ cd house_finder
  $ pip install -r requirements.txt
  ```
  
## usage 

 move to the directory that you installed this app.

  ```
  $ cd hosue_finder
  ```

run the app

  ```
  $ python main.py
  ```
